# 🩺 ד"ר קלוד — רופא משפחה דיגיטלי

אפליקציית PWA לניהול רפואי אישי מבוססת Anthropic AI.

## ✨ פיצ'רים

- 💬 **ייעוץ AI** — שיחה עם ד"ר קלוד, שאלות, אבחנה ראשונית
- 📷 **ניתוח תמונות** — עור, גרון, עין, פריחה, פצע
- 📄 **קריאת מסמכים** — PDF, תמונות מסמכים, תוצאות בדיקות
- 📁 **תיק רפואי** — פרופיל, מחלות, תרופות, אלרגיות
- 📊 **סיכום AI** — סיכום קליני מלא של התיק
- 💊 **מעקב תרופות** — לקיחה יומית, 7 ימים אחרונים
- 🧪 **מעקב בדיקות** — תזמון וסימון ביצוע
- 📅 **ביקורות מעקב** — תזמון פגישות
- 🔄 **שמירה אוטומטית** — כל הנתונים ב-localStorage

## 🚀 התקנה מהירה

### אופציה 1 — GitHub Pages (מומלץ)

1. פורק (Fork) את הריפוזיטורי הזה
2. עבור לـ **Settings → Pages**
3. תחת **Source** בחר **Deploy from a branch**
4. בחר **main** ולחץ **Save**
5. האפליקציה תהיה זמינה בכתובת:<br>
   `https://YOUR-USERNAME.github.io/dr-claude`

### אופציה 2 — הורדה ישירה

1. הורד את קובץ `index.html`
2. שמור לאפליקציית **Files** באייפון
3. פתח עם **Safari**
4. לחץ **שתף ⬆️ → הוסף למסך הבית** להתקנה כאפליקציה

## 🔑 הגדרת מפתח API

1. היכנס ל-[console.anthropic.com](https://console.anthropic.com)
2. צור חשבון חינמי (כולל $5 קרדיט)
3. לחץ **API Keys → Create Key**
4. העתק את המפתח (`sk-ant-...`)
5. פתח את האפליקציה → **הגדרות ⚙️** → הדבק ושמור

## 📱 התקנה כאפליקציה

## ⚠️ הבהרה רפואית

ד"ר קלוד מספק מידע רפואי **ראשוני בלבד**.  
אינו מחליף ביקור פיזי אצל רופא מוסמך.  
במצב חירום — **התקשר מיד ל-101 (מד"א)**.

## 🛠️ טכנולוגיות

- Vanilla JavaScript (ללא frameworks)
- Anthropic Claude API (`claude-haiku-4-5-20251001`)
- PDF.js לצפייה ב-PDF
- localStorage לשמירת נתונים
- PWA — ניתן להתקנה על מסך הבית
